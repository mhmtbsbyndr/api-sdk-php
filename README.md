# Garpa.de Onlineshop - API SDK for PHP

Install
PHP client for <a href="https://www.garpa.de">Garpa.de</a> Onlineshop API.  

via Composer
$ composer require mhmtbsbyndr/api-sdk

Via GitHub
$ git clone git@github.com:mhmtbsbyndr/api-sdk-php.git
